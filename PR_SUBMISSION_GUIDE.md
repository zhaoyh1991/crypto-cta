## 🔧 部署说明

### 环境要求
- Python 3.8+
- 至少2GB可用内存
- 网络连接（用于数据获取）

### 安装步骤
```bash
# 1. 进入项目目录
cd crypto_cta

# 2. 安装依赖
./install_deps.sh

# 3. 验证安装
python -c "import pandas; import numpy; print('依赖安装成功')"
```
